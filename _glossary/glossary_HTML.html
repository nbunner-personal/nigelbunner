--- layout: glossary title: What does HTML mean? permalink: /glossary/hper-text-markup-language.html ---

The programming language used to display the content you see on your webpage. It stands for Hyper Text Markup Language. If you choose View Source from your Web Browser's menu options you'll see the HTML code that makes up this page. All the HTML code used on the websites I design is handcoded to ensure it remains clean and Standards Compliant.

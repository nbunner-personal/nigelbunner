--- layout: glossary title: What is a web browser? permalink: /glossary/glossary_Web-Browser.html ---

A web browser is a piece of software on your computer that you use to view pages from the internet. If you're reading this, your using a web browser right now.

There are various different browsers around. The most commonly used is Internet Explorer but many people use Firefox, Chrome, Opera and Safari for the advantages such as increased speed and extra security that these browsers may offer to the user.
